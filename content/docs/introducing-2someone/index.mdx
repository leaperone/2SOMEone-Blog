## Core Products

### Lixi - Live Stream Data Analytics Expert

Lixi is a professional live stream data analytics tool that provides Bilibili gift statistics and Captain (Da Hang Hai) data analysis, offering deep insights into fan spending behavior.

#### Address System
Manage fan addresses for easier gift delivery. Supports address collection and export functions, making it convenient for streamers to manage fan benefits distribution.

#### Captain Spending Statistics
Provides complete Captain spending records that can be matched with addresses for export, easily tracking fan support.

#### Captain Roster Review
Provides Captain roster lists, Captain statistics, with address matching for export, facilitating fan maintenance and appreciation.

#### User Threshold Filtering
Filter users based on spending amount, interaction frequency, and other criteria to precisely identify core fan groups.

#### Anonymous Submissions

#### Public Posts
Let viewers browse everyone's submissions, increasing interactivity and fun, creating a community atmosphere.

#### Interactive Community
Build an exclusive fan community to enhance interaction with fans and increase fan loyalty.

#### Danmaku System Integration
