# 2SOMEone - Универсальный инструмент управления для стримеров

**Люди, Стриминг, Жизнь**

2SOMEone — это универсальный инструмент управления, разработанный для стримеров Bilibili и Douyin, объединяющий такие ключевые функции, как личная страница, ящик вопросов, плейлист и управление фанатами, чтобы помочь стримерам более эффективно управлять прямыми трансляциями и взаимодействием с фанатами.

В настоящее время **8 523 стримера** любят эту платформу, с общим количеством **88 465 сообщений** в Bubble Tree Hole.

## Основные продукты

### Lixi - Эксперт по аналитике данных стриминга

Lixi — это профессиональный инструмент аналитики данных прямых трансляций, предоставляющий статистику подарков Bilibili и анализ данных Captain (Da Hang Hai), предлагая глубокое понимание поведения фанатов.

#### Система адресов
Управляйте адресами фанатов для более удобной доставки подарков. Поддерживает функции сбора и экспорта адресов, упрощая управление распределением бонусов для фанатов.

#### Статистика расходов Captain
Предоставляет полные записи расходов Captain, которые можно сопоставить с адресами для экспорта, легко отслеживая поддержку фанатов.

#### Обзор списка Captain
Предоставляет списки Captain и статистику с сопоставлением адресов для экспорта, облегчая поддержку и благодарность фанатам.

#### Фильтрация пользователей по порогу
Фильтруйте пользователей по сумме расходов, частоте взаимодействия и другим критериям для точного определения основных групп фанатов.

### Bubble Tree Hole - Новый опыт анонимного взаимодействия

Bubble Tree Hole предоставляет новый способ анонимного взаимодействия между стримерами и фанатами:

#### Анонимные сообщения
Защищает конфиденциальность фанатов, позволяя отправлять каждое сообщение безопасно, не беспокоясь о раскрытии личности.

#### Умный AI-помощник
AI автоматически проверяет контент, анализирует эмоции и интеллектуально генерирует предложения ответов, снижая нагрузку на модерацию стримера.

#### Публичные посты
Позвольте зрителям просматривать сообщения всех, увеличивая интерактивность и веселье, создавая атмосферу сообщества.

#### Интерактивное сообщество
Создайте эксклюзивное сообщество фанатов для улучшения взаимодействия с фанатами и повышения их лояльности.

#### Интеграция системы данмаку
Интегрированная система данмаку создаёт иммерсивный опыт стриминга, с умным управлением сообщениями, поддерживающим категоризированное архивирование.

### Wingman - Профессиональная машина данмаку

Wingman — это профессиональный инструмент анализа данмаку, который помогает стримерам лучше понимать взаимодействие аудитории, оптимизировать контент трансляций и улучшать результаты стриминга.

#### Анализ данмаку
Мониторинг и анализ данных данмаку в реальном времени, получение информации о трендах взаимодействия аудитории и горячих темах.

#### Отслеживание подарков
Отслеживайте данные Super Chat и подарков, чтобы понять поддержку фанатов и источники дохода.

#### Анализ поведения пользователей
Анализируйте поведение взаимодействия пользователей для оптимизации контента трансляций и стратегий взаимодействия.

#### Управление Captain
Управляйте и отслеживайте данные Captain для поддержания основных групп фанатов.

### Личная страница - Покажите свой творческий стиль

Личная страница позволяет стримерам централизованно отображать свой контент и информацию:

- **Настраиваемый редактор профиля**: Создайте уникальный личный образ
- **Система управления плейлистами**: Поделитесь своим музыкальным вкусом
- **Адаптивный дизайн**: Идеальная адаптация к различным устройствам
- **Поделиться в один клик**: Расширьте своё влияние

Личная страница объединяет избранные видео, новости, взаимодействие с фанатами и расписание трансляций, предоставляя фанатам единый портал для знакомства со стримером.

## Зона гильдии

Предоставляет комплексные решения для операций гильдии, помогая гильдиям достичь профессиональных операций и эффективного управления.

### Страница гильдии
Создайте единый портал для членов гильдии, публично отображая карточки участников, домашние страницы, tree hole, плейлисты и другие каналы.

### Управление участниками
Ищите аккаунты, добавляйте участников, настраивайте сортировку или удаляйте участников в панели управления с синхронизацией в реальном времени на публичную страницу.

### Безопасный обмен
Копируйте ссылку `/club/slug` в один клик, страница по умолчанию не отображается на фронтенде, видна только посетителям, знающим адрес.

### Инструменты скаута

#### Автоматизированный запрос контракта
Запрос статуса контракта в один клик, чтобы понять ситуацию с подписанием стримеров и быстро определить потенциальные цели для сотрудничества.

#### Экспорт записей контрактов
Пакетный экспорт данных контрактов в различных форматах для удобного управления и анализа данных.

#### Управление скаутами
Находите стримеров, проверяйте статусы, управляйте экспортом — универсальные инструменты скаута для роста бизнеса.
